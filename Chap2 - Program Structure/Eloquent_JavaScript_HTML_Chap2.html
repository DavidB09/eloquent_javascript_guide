Looping a Triangle

Question

Write a loop that makes seven calls to console.log to output the following triangle:

#    
##  
###
####
#####
######
#######

Personal Answer

let answer = "#";
for (let i = 0; i < 7; i++) {
    console.log(answer); 
    answer += "#"; 
}

Line 1 binds answer to the character in the first row of the answer.

Line 2 - Line 5 introduce a for loop that checks if i is less than the number of rows in the answer. If the < comparison is true, it prints out the current answer and appends the hash sign character to the answer. Then, 1 is added to i and the loop continues to its next iteration.

Author's Answer

for (let line = "#"; line.length < 8; line += "#")
console.log(line);

The author's answer also uses the for loop but more concisely.

Line 1 - Line 2 binds line to the character in the first row of the answer. While the length of line is less than the intended number of hash marks in the final row of the answer, a hash sign character is added to the string and the loop continues to its next iteration.

FizzBuzz

Question

Write a program that uses console.log to print all the numbers from 1 to 100, with two exceptions. For numbers divisible by 3, print "Fizz" instead of the number, and for numbers divisible by 5 (and not 3), print "Buzz" instead.

When you have that working, modify your program to print "FizzBuzz" for numbers that are divisible by both 3 and 5 (and still print "Fizz" or "Buzz" for numbers divisible by only one of those).

Personal Answer

for (let i = 1; i <= 100; i++) {
    if (i % 3 == 0) {
        if (i % 5) {
            console.log("FizzBuzz")
        } else {
            console.log("Fizz"); 
        }
    } else if (i % 5 == 0) {
        console.log("Buzz"); 
    } else {
        console.log(i); 
    }
}

Line 1 creates a for loop that binds i to 1 and checks if it's value is less than or equal to 100. While the < comparison is true, the loop continues through it's next statements before moving on to the next iteration.

Line 2 checks if i is divisible by 3 with the remainder of 0 with a conditional statement.

If Line 2 is true, then Line 3 - 7 checks if i is also divisible by 5 with the remainder of 0. If true, then FizzBuzz is printed to the console. Else, Fizz is printed.

If Line 2 is false, then Line 8 - 9 checks if i is divisible by 5 with the remainder of 0. If true, then Buzz is printed on the console.

If all of the previous conditions were false, then i, or the regular number, is printed to the console.

Author's Answer

for (let n = 1; n <= 100; n++) {
    let output = "";
    if (n % 3 == 0) output += "Fizz";
    if (n % 5 == 0) output += "Buzz";
    console.log(output || n);
}

Line 1 creates a for loop that binds n to 1 and checks if it's value is less than or equal to 100. While the < comparison is true, the loop continues through its next statements before moving on to the next interation.

Line 2 binds output to an empty string.

Line 3 uses a conditional statement to check whether n is divisible by 3 with a remainder of 0. If true, then the string Fizz is added to output.

Line 4 uses a conditional statement to check whether n is divisible by 5 with a remainder of 0. If true, then the string Buzz is added to output.

Line 5 uses an "or" operator to check whether output is an empty string. If output has no value, then n is printed to the console. Else, output is printed to the console.

Chessboard

Question

Write a program that creates a string that represents an 8Ã—8 grid, using newline characters to separate lines. At each position of the grid there is either a space or a "#" character. The characters should form a chessboard.

 # # # #
# # # # 
 # # # #
# # # # 
 # # # #
# # # # 
 # # # #
# # # #

Personal Answer

let binding = 8; 
let finalString = ""; 
for (let a = 1; a <= binding; a++) {
    for (let b = 1; b <= binding; b++) { 
        if (a % 2 == 0) {
            b % 2 == 0 ? finalString += " " : finalString += "#"; 
        } else {
            b % 2 == 0 ? finalString += "#" : finalString += " "; 
        }
    }
    finalString += "\n"; 
}     
console.log(finalString);

Line 1 - Line 2 binds the length and width of the grid to binding and initializes finalString to an empty string.

Line 3 creates a for loop that is intended to represent each row of the matrix. While a is less than binding, the loop executes its next statements before moving on to the next row.

Line 4 also creates a for loop that represents the characters in a particular row. While b is less than binding, a particular character is added to finalString before moving on to the next character.

Line 5 checks whether the current row is divisible by 2 with a remainder of 0, or whether it is even or odd.

If Line 5 is true, Line 6 uses an abbreviated version of the if conditional statement to check whether the current character is even or odd. If the < comparison preceding the ? character is true, then an empty space or " " is added to finalString. Else, a hash sign or "#" is added to finalString.

Since the all of the even chessboard rows begin with a hash sign, the condition statement results in Line 8 are switched whenever Line 5 is false. Now, if the < comparison preceding the ? character is true, an hash sign is added finalString. When false, an empty space is added.

Line 11 adds a newline character to fnialString whenever a row is completed, causing the next row to be on a new line instead of one long continous string.

Author's Answer

let size = 8;
let board = "";
for (let y = 0; y < size; y++) {
    for (let x = 0; x < size; x++) {
        if ((x + y) % 2 == 0) {
            board += " ";
        } else {
            board += "#";
        }
    }
    board += "\n";
}
console.log(board);

Line 1 - Line 2 binds the length and width of the grid to size and initializes board to an empty string.

Line 3 creates a for loop that is intended to represent each row of the matrix. While y is less than size, the loop executes its next statements before moving on to the next row.

Line 4 also creates a for loop that represents the characters in a particular row. While x is less than size, a particular character is added to board before moving on to the next character.

Line 5 uses a conditonal statement to check whether the values of x and y added together is even or odd.

If Line 5 is true and the result is even, Line 6 adds an empty space character to board.

Else, Line 8 adds a hash sign character when x and y added together results in an odd value.

Line 11 adds a newline character to board whenever a row is completed.
