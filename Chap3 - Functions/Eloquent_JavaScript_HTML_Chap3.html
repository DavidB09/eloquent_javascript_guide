Chapter 3, "Functions"

Minimum

Question

Write a function min that takes two arguments and returns their minimum.

Personal Answer

function min(num1, num2) {
    if (num1 < num2)
        return num1; 
    else  
        return num2; 
}

Line 1 declares min as a function with two parameters.

Line 2 - Line 3 uses an if statement that checks whether num1 is smaller than num2. If the < comparison is true, min returns num1.

If Line 2 is false, then min returns num2.

Author's Answer

function min(a, b) {
    if (a < b) return a; 
    else return b; 
}

The author's answer uses similiar code just without indention and different parameter names.

Recursion

Question

We’ve seen that % (the remainder operator) can be used to test whether a number is even or odd by using % 2 to see whether it’s divisible by two. Here’s another way to define whether a positive whole number is even or odd:

- Zero is even.

- One is odd.

- For any other number N, its evenness is the same as N - 2.

Define a recursive function isEven corresponding to this description. The function should accept a single parameter (a positive, whole number) and return a Boolean.

Personal Answer

function isEven(number) {
    if (number < 0) 
        return isEven(number + 2); 
    else if (number == 0)
        return true; 
    else if (number == 1)
        return false; 
    else 
        return isEven(number - 2); 
}

Line 1 declares isEven as a function with one parameter.

Line 2 - 3 checks if number is negative. If true, then isEven is rerun with a new parameter equal to number + 2 since the number will need to equal either 1 or 0.

Line 4 - 5 checks if number is equal to 0. If true, then number is even and the Boolean value true is returned.

Line 6 - 7 checks if number is equal to 1. If true, then number is odd and the the Boolean value false is returned.

If all other statements are false, then Line 8 - 9 recalls isEven with a new parameter equal to number - 2 since the number will need to equal either 1 or 0.

Author's Answer

function isEven(n) {
    if (n == 0) return true; 
    else if (n == 1) return false; 
    else if (n < 0) return isEven(-n); 
    else return isEven(n - 2); 
}

The author's answer uses similar code with Line 1 correlating with Line 1, Line 2 with Line 4 - 5, Line 3 with Line 6 - 7, and Line 5 with Line 8 - 9.

Line 4 uses a different approach with negative numbers. Instead of adding 2 to the number, the new parameter for isEven is n reversed and made positive.

Bean Counting

Question

Part I:

Write a function countBs that takes a string as its only argument and returns a number that indicates how many uppercase "B" characters there are in the string.

Part II:

Next, write a function called countChar that behaves like countBs, except it takes a second argument that indicates the character that is to be counted (rather than counting only uppercase “B” characters).

Personal Answer

Part I:

function countBs(string) {
    let count = 0; 
    for (let i = 0; i < string.length; i++) {
        if (string[i] == "B") count++; 
    }     
    return count;
}

Part II:

function countChar(string, character) {
    let count = 0; 
    for (let i = 0; i < string.length; i++) {
        if (string[i] == character) count++; 
    }     
    return count;
}

Part I:

Line 1 creates the function countBs with string as its only parameter.

Line 2 initializes count to 0.

Line 3 creates a for loop where i represents one specific character in string. While i is less than string's length, it moves on to the next character.

Line 4 checks whether the current charater in string is equal to uppercase "B". If so, then count is increased by one.

Line 5 returns the final count value.

Part II:

countChar uses the same code just with an extra parameter called character to refer to the particular letter being referenced in the comparison.

Author's Answer

Part II:

function countChar(string, ch) {
    let counted = 0;
    for (let i = 0; i < string.length; i++) {
        if (string[i] == ch) {
            counted += 1;
        }
    }
    return counted;
}

Part I:

function countBs(string) {
    return countChar(string, "B");
}

Part II:

The author uses the same code just with different variable names and expressions.

Part I:

The author just uses countBs to call the countChar function with the ch parameter set to the character "B" in Line 2
